# Análisis de Ventas

## Objetivo
Analizar datos de ventas para identificar tendencias, productos rentables y oportunidades de mejora.

## Herramientas
- Python (Pandas, NumPy)
- Excel
- Visualización de datos

## Actividades realizadas
- Limpieza de datos
- Análisis exploratorio
- Generación de métricas clave
- Visualización básica

## Resultado
Se identificaron patrones de ventas y productos con bajo margen.
